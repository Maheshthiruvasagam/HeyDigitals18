# HeyDigitals18

A modern digital services agency website built with React and Tailwind CSS.

## Features

- Responsive design
- Modern UI/UX
- Service showcase
- Team portfolio
- Contact form

## Tech Stack

- React
- React Router
- Tailwind CSS
- React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
